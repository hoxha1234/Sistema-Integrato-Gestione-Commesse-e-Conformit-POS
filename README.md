# Sistema-Integrato-Gestione-Commesse-e-Conformit-POS
SafeSite è un software gestionale in Laravel 11 progettato per la sicurezza nei cantieri. Monitora l'avanzamento delle commesse tramite una dashboard Kanban, automatizza la validazione dei POS, gestisce le scadenze della formazione operaia e blocca i lavori in caso di irregolarità documentale. Sicurezza e conformità in un click.
